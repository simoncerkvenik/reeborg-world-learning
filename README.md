# Reeborg's World Learning

This repository documents my learning process with Reeborg's World.

The goal is not only to solve the exercises, but to write down the thinking process behind each solution.

I will document everything step by step:
- what the task was
- how I first understood the problem
- what confused me
- which mistakes happened
- how I debugged the robot's behavior
- what logic finally worked
- what I learned from the exercise

This repository is part of my programming learning journey.  
It focuses on understanding logic, conditions, loops, debugging, and algorithmic thinking.
